HTML-Element

<html>
Omsluter alla taggar och talar om för webbläsaren att det är HTML-kod.
<!-- -->
Används för kommentarer i HTML.
<head>
Innehåller information till webbläsaren och sökmotorer.
<meta>
Används för meta-data.
<title>
Anger hemsidans titel.
<body>
Innehåller alla element som är synliga på webbsidan.
<h1> - <h6>
Huvudrubrik och underrubriker.
<p>
Definerar ett stycke.
<hr>
Används när man byter tema på sidan, normalt sett visas också en horisontell linje.
<br>
Ny rad.
<b>
Fet text.
<i>
Kursiv text.
<em>
Används för att betona text.
<strong>
Används för extra viktig text.
<sup>
Höjer upp text.
<sub>
Sänker ner text.
<u>
Understruken text.
<small>
Mindre text.
<s>
Genomstruken text.
<blockquote>
Citat från andra källor.
<cite>
Används för att definera titlar på ex. filmer.
<q>
Används för citat, lägger automatiskt till "".
<mark>
Markerad text (märkpenna).
<address>
Anger att texten är en adress, kan vara vanlig adress eller e-post.
<del>
Används för att visa borttagen text.
<ins>
Används för att visa infogad text.
<abbr>
Används för förkortningar.
<ul>
Punktlista (unordered list)
<ol>
Numbrerad lista (ordered list)
<li>
Listelement
<dl>
Definitionslista
<dt>
Definitionsterm
<dd>
Definitionsbeskrivning
<a>
Hyperlänk
